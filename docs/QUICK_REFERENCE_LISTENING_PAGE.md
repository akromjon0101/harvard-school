## 🎯 WHAT'S WHERE

### **Header (Dark Gray)**
- 🔴 Red "SECTION 1/4" badge → Shows current section
- ⏱️ Blue "00:30:00" timer → Time remaining
- 🔴 Red "FINISH" button → Quit test (top right)

### **Audio Player (Sticky - Stays at Top)**
- 🎧 Official IELTS instruction text
- 🎵 Play / Pause / Volume controls
- 📊 Progress bar
- ⏱️ Current time / Duration

### **Left Panel (Questions)**
- 📋 White background
- 🔵 Blue question numbers in circles
- 📝 Question text
- 📸 Picture previews (for picture choice)
- 🔤 Multiple choice options (for MCQ)
- 📌 Scrollable to see all questions

### **Right Panel (Answer Sheet)**
- 📄 Cream/off-white background (paper color)
- 🟫 Gold/brown border (paper style)
- ✏️ Underlines for writing answers
- 🔵 Radio buttons for selecting answers
- ☑️ Checkboxes for multiple selections
- ⚠️ "Write in CAPITAL LETTERS" instruction
- 📌 Scrollable to see all answers

### **Footer (White)**
- 🔵 "NEXT SECTION (Part 2) →" button (blue)
- After section 4: 🟢 "SUBMIT TEST" button (green)
- Centered at bottom of page

## 🖱️ HOW TO INTERACT

### **For Picture Choice (Q1)**
```
LEFT (Read):                RIGHT (Answer):
1️⃣ Choose the image:        1  [○A] [○B] [○C]

[Image A]  [Image B]           ↑ Click one circle
[Image C]
                            What happens:
                            - Circle fills with blue
                            - Answer is saved
                            - Can change anytime
```

### **For Short Answer (Q2)**
```
LEFT (Read):                RIGHT (Answer):
2️⃣ How many people?         2 ________________
                               ↑ Click and type

                            What happens:
                            - Cursor appears
                            - Type your answer
                            - Auto-capitalizes
                            - Max 80 characters
```

### **For Matching (Q3)**
```
LEFT (Read):                RIGHT (Answer):
3️⃣ Match the items:        3 [Select option ▼]
A) Teacher                     ↑ Click dropdown
B) Engineer
C) Doctor                   Options appear:
                            A) Teacher
                            B) Engineer
                            C) Doctor
```

### **For MCQ Multiple (Q4)**
```
LEFT (Read):                RIGHT (Answer):
4️⃣ Select correct items:    4 [☐A] [☐B]
                               [☐C] [☐D]
A) Item 1                      ↑ Click to check
B) Item 2
C) Item 3                   What happens:
D) Item 4                   - Multiple can be selected
                            - Checkmarks appear
                            - Can change anytime
```

## ⏱️ TIMER BEHAVIOR

```
Start of test:     00:30:00 ← 30 minutes
After 1 min:       00:29:00 ← Counts down
...
After 25 min:      00:05:00 ← 5 minutes left
After 29 min:      00:01:00 ← 1 minute left
Reaches zero:      00:00:00 ← Time's up!
```

**Timer:**
- ⏱️ Shows in blue in header
- Updates every second
- Counts down from 30 minutes
- Doesn't stop (continues through all sections)
- For display only (doesn't auto-submit)

## 🎬 WORKFLOW EXAMPLE

### **Student Opens Test:**
```
1. Click "Listening" menu
2. See test list
3. Click "Start Test" on desired test
4. Click "Start Listening Test" button
→ ENTERS SECTION 1
```

### **Student Takes Section 1:**
```
5. See Questions 1-5 on LEFT
6. See blank Answer Sheet on RIGHT
7. PLAY audio and LISTEN
8. READ questions while listening
9. ANSWER questions on right panel:
   - Click radio buttons for MCQ
   - Type in underlines for text
   - Select from dropdowns for matching
10. SCROLL down to answer more
11. Continue answering all questions
→ CLICK "NEXT SECTION"
```

### **Student Takes Sections 2-3-4:**
```
12. Repeat process for Section 2
13. Repeat process for Section 3
14. Repeat process for Section 4
→ CLICK "SUBMIT TEST"
```

### **Test Ends:**
```
15. See "Test Submitted!" message
16. Answers recorded
17. Can return to dashboard
```
