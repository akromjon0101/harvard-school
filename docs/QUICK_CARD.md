## 2️⃣ PASTE & CREATE TOOL STEPS

```
Step 1: Click [📋 Paste & Create]
       ↓
Step 2: Paste text (no [input] tags!)
       ↓
Step 3: Click [Next: Add Inputs →]
       ↓
Step 4: Click where you want blank → Click [➕ Add Input Here]
        Repeat for all blanks
       ↓
Step 5: Click [📊 Create Questions]
       ↓
Step 6: Fill Answer field for each question
       ↓
Step 7: Click [✅ Add All Questions]
       ↓
✨ Done! All questions added!
```

## 6️⃣ TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Can't find button | Make sure Question Type is "Gap Filling / Completion" |
| Button not showing | Only appears for Completion/Gap Filling questions |
| Text won't open modal | Click the button again |
| Can't add inputs | Click in text area first, THEN click button |
| Questions not appearing | Make sure you filled in the Answer field |
