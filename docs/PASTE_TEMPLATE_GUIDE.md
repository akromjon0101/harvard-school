1. **Count `[input]` tags** in your text
2. **Match with answers** (one per line)
3. **Auto-number questions** starting from your chosen number
4. **Create full question text** from the template lines

## ✅ Best Practices

### ✓ DO:
- Replace every blank with `[input]`
- One answer per line
- Keep answers short (1-3 words)
- Start with question number (e.g., 31, 1, etc.)

### ✗ DON'T:
- Leave blanks as `____` or `_____`
- Add extra spaces in answers
- Mix `[input]` with other markers
- Copy formatting from Word/PDF (paste as plain text)

## 🎯 Common Mistakes & Fixes

### ❌ Mistake 1: Blanks Not Marked
```
Original: "The building was built in __________."
Use this: "The building was built in [input]."
```

### ❌ Mistake 2: Answers Not Aligned
```
Wrong:
[input] [input] [input]
rock, brother, tools

Right:
[input] [input] [input]
rock
brother
tools
```

### ❌ Mistake 3: Extra Spaces in Answers
```
Wrong: "  rocks  " (extra spaces)
Right: "rocks" (no spaces)
```

### ❌ Mistake 4: Skipped Blanks
```
Wrong: Only 8 answers for 10 [input] tags
Right: Count [input] tags first, then provide exact number of answers
```
