## 🎯 HOW EACH QUESTION TYPE APPEARS

### **1. Picture Choice Questions**
```
LEFT (Questions):                RIGHT (Answer Sheet):
─────────────────────────────────────────────────────
1 Which image matches          1 [ A ]  [ B ] [ C ]
   the description?

[Picture: 70x70px A]           • Radio buttons to select A, B, or C
[Picture: 70x70px B]           • Shows as circles in answer sheet
[Picture: 70x70px C]
```

### **2. Multiple Choice - Single Answer**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
2 What is the main topic?      2 [ A ]  [ B ] [ C ] [ D ]

A) Environment
B) Technology                   • Select ONE answer
C) Education                    • Shows as radio buttons
D) Healthcare
```

### **3. Multiple Choice - Multiple Answers**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
3 Which statements are         3 [ ☑ A ] [ ☐ B ]
   correct?                       [ ☐ C ] [ ☐ D ]

A) The course costs £500
B) Registration is in April    • Can select multiple
C) It includes materials       • Shows as checkboxes
D) Online access only
```

### **4. Short Answer**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
4 How many students attended?  4 _____________________

                                • Underline for writing
                                • Auto-capitalized
                                • Text input field
```

### **5. Form/Note Completion**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
5 Complete the form.            5 _____________________
   Use NO MORE THAN 2 WORDS.
                                • Write answer here
Name: ___________
Age: ___________                • Limited to 80 chars
```

### **6. Matching**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
6 Match descriptions to jobs   6 [Select... ▼]
                                 Shows dropdown with:
A) Teaching                      A) Teaching
B) Engineering                   B) Engineering
C) Marketing                      C) Marketing
D) Nursing

1. Works with computers        • Dropdown to select match
2. Works with people's health
```

### **7. Classification**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
7 Which category?              7 [Select category ▼]
   YES / NO / NOT GIVEN
                                 Options:
A) Statement 1                   - YES
B) Statement 2                   - NO
                                 - NOT GIVEN
```

### **8. Diagram/Map Labelling**
```
LEFT:                           RIGHT:
─────────────────────────────────────────────────────
8 Label the diagram            8 _____________________

[Image: 70x70px diagram]         • Write label name
                                 • For: Room A, Entrance, etc.
```

## 📝 EXAMPLE WALKTHROUGH

### **Student's Perspective:**

1. **Starts test** → Sees intro page with "Start Listening Test" button
2. **Clicks Start** → Enters Section 1 with:
   - Audio player at top (sticky)
   - Questions 1-5 on left panel
   - Answer sheet on right panel
   - Blank underlines ready to write
3. **Reads questions** while audio plays
4. **Listens to audio** and writes answers on right side
5. **Clicks underline** and types answer → **AUTO-CAPITALIZED**
6. **For MCQ** → Clicks A/B/C radio buttons on right panel
7. **For pictures** → Clicks image circles
8. **Clicks "NEXT SECTION"** → Goes to Section 2 with new questions
9. **After Section 4** → Clicks "SUBMIT TEST" button
10. **Sees confirmation** → "Test Submitted!"

## 🐛 IF SOMETHING LOOKS WRONG

**The new CSS file is:** `/src/styles/listening-ielts-style.css`

**If you see old styling:**
1. Hard refresh: **Cmd+Shift+R**
2. Clear browser cache
3. Close and reopen browser tab

**If colors are wrong:**
- Answer sheet should be cream/off-white
- Header should be dark gray/black
- Numbers should be blue circles
- Buttons should be blue or green

**If layout is broken:**
- Questions should be on LEFT
- Answer sheet should be on RIGHT
- On mobile: stacked vertically
- Audio player should stay at top when scrolling
