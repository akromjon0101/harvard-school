## What Happens When You Use It

### 1️⃣ Fill the Quick Builder Form
```
No.           → 1
Question Text → "The company was founded in [input]"
Answer        → "1995"
```

### 2️⃣ Click "➕ Add" Button
```
✓ Input cleared
✓ Question added to list
✓ Count increases: "Questions Added: 3"
```

### 3️⃣ See Question in List
```
✓ 1. The company was founded in [input] [completion]
```

### 4️⃣ Repeat for More Questions
```
No.           → 2
Question Text → "The building had [input] floors"
Answer        → "20"
Click ➕ Add
→ ✓ 2. The building had [input] floors [completion]
```
