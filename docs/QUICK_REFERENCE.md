## What You Can Create

✅ **Gap Filling** - Type words in blanks  
✅ **Multiple Choice** - Choose A/B/C/D  
✅ **True/False/Not Given** - T/F/NG format  
✅ **Matching** - Connect related items  
✅ **Labelling** - Label image/map parts  

---

## Quick Tips

📝 **Tip 1:** Number questions 1-10, 11-20, etc.  
📝 **Tip 2:** Be specific with answers (exact words)  
📝 **Tip 3:** Click × to delete a question  
📝 **Tip 4:** Can edit full details in editor below  
📝 **Tip 5:** Click "PUBLISH OFFICIAL EXAM" when done

## Troubleshooting (30 seconds)

| Problem | Solution |
|---------|----------|

| Questions not showing | Click ➕ Add button |

## Common Tasks

### Create 10 Gap-Filling Questions
```
1. Add new group → Gap Filling
2. Repeat 10 times:
   - No. [1-10]
   - Question: ______
   - Answer: ____
   - Click ➕ Add

```
