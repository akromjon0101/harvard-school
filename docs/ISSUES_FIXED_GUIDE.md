## 🚀 HOW TO USE NEW PASTE TOOL:

### Step 1: Click "📋 Paste & Create" Button

Look for the purple button in your Listening/Reading section:

```
[📋 Paste & Create] or [➕ Add]
```

---

### Step 2: Paste Your Text (WITHOUT blanks)

**Step 1 Modal:**
```
┌─────────────────────────────────────────────┐
│  📋 Step 1: Paste Your Text                 │
│                                              │
│  Just paste your text here. We'll help you  │
│  add [input] blanks and answers next.       │
│                                              │

│  │ realised that stones beneath the     │  │

│                                              │
│  [Next: Add Inputs →]  [Cancel]            │
└─────────────────────────────────────────────┘
```

Just paste **plain text** - NO [input] tags needed yet!

---

### Step 3: Click Where You Want Blanks

**Step 2 Modal:**

```
┌─────────────────────────────────────────────┐
│  📋 Step 2: Add Inputs & Answers            │
│                                              │
│  📌 Click in the text where you want to     │
│     add a blank → Click "+ Add Input Here"  │
│                                              │
│  [➕ Add Input Here]  [📊 Create Questions] │
│                                              │
│  ┌──────────────────────────────────────┐  │
│  │ In the 1930s, a local teacher        │  │
│  │ realised that stones beneath the     │  │
│  │ bog surface were once [input]. His   │  │
│  │ [input] became an archaeologist...   │  │
│  └──────────────────────────────────────┘  │
│                                              │
│  Start Question Number: [31]                │
│                                              │
│  (Scroll down to see Questions section)     │
└─────────────────────────────────────────────┘
```

**HOW TO ADD BLANKS:**
1. Click in the textarea where you want a blank
2. Click **"➕ Add Input Here"** button
3. The word `[input]` appears at your cursor position
4. Repeat for all blanks

---

### Step 4: Click "Create Questions"

After adding all [input] blanks, click **"📊 Create Questions"** button.

The system automatically creates questions from your text!

```
✏️ Questions & Answers (10)

Start Question Number: [31]

[Question 1]
No.: 31
Text: In the 1930s, a local teacher realised that stones 
      beneath the bog surface were once [input].
Answer: [____________] ← Fill this in!

[Question 2]
No.: 32
Text: His [input] became an archaeologist and undertook...
Answer: [____________] ← Fill this in!

[Question 3]
No.: 33
...
```

---

### Step 5: Fill in Answers

For each question, type the **correct answer** in the Answer field.

```
Question 1 → Answer: rocks
Question 2 → Answer: brother
Question 3 → Answer: stone tools
...
```

---

### Step 6: Click "✅ Add All Questions"

All questions are instantly added to your exam! 🎉

```
✅ Added 10 questions!
```
