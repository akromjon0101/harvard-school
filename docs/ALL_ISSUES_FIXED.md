### Steps to Create 5 Questions:

**1. Click "📋 Paste & Create"**
- Modal opens (Step 1)

- No [input] tags needed!
- Just plain text

**3. Click "Next: Add Inputs →"**
- Modal changes to Step 2
- Text appears in textarea

**4. Click to add [input] blanks**
```
Position your cursor at:
  "...were once |rocks" → click "➕ Add Input Here" → becomes [input]
  "...His |brother" → click "➕ Add Input Here" → becomes [input]
  "...to dig |stone tools" → click → becomes [input]
  "...Neolithic|." → click → becomes [input]
```

Result:
```
In the 1930s, a local teacher realised that stones 
beneath the bog surface were once [input]. His [input] 
became an archaeologist and undertook an investigation 
of the site. A traditional method used by local people 
to dig [input] was used to identify where stones 
were located. Carbon dating later proved the site was 
[input].
```

**5. Click "📊 Create Questions"**
- System finds all [input] blanks
- Creates 4 questions automatically

**6. Fill in Answers**
```
Question 31: In the 1930s... [input]. 
             Answer: rocks ✓

Question 32: His [input] became...
             Answer: brother ✓

Question 33: ...to dig [input]...
             Answer: stone tools ✓

             Answer: Neolithic ✓
```

**7. Click "✅ Add All Questions"**

### ❌ COMMON MISTAKES:

**Mistake 1:** Don't copy formatted text from Word/PDF
```
❌ Copy from PDF/Word (has extra formatting)
✅ Copy plain text from .txt files
```

**Mistake 2:** Don't edit text during Steps
```

```
