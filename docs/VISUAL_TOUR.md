### 🎯 How to Use the Quick Builder

#### Input Form

```
┌────────────────────────────────────────────────────┐
│ 📝 Easy Question Builder                          │
├────────────────────────────────────────────────────┤
│                                                    │
│ No.:  [_________]                                  │
│       Enter question number (1, 2, 3, etc.)       │
│                                                    │
│ Question Text: [___________________]               │
│       e.g., "The building was constructed in _"   │
│                                                    │
│ Answer: [_________]                                │
│       e.g., "1995" or "Tokyo" or "vital"          │
│                                                    │
│              [➕ ADD QUESTION]                     │
│                                                    │
└────────────────────────────────────────────────────┘
```

#### Step-by-Step Example

```
STEP 1: Fill No.
┌─────────────────┐
│ No.: │1        │  ← Type or paste "1"
└─────────────────┘

STEP 2: Fill Question Text
┌──────────────────────────────────────┐
│ Question: │The city was... _________ │  ← Type the sentence
└──────────────────────────────────────┘

STEP 3: Fill Answer
┌──────────────────┐
│ Answer: │Tokyo   │  ← Type the correct answer
└──────────────────┘

STEP 4: Click Add
┌──────────────────┐
│ [➕ ADD QUESTION]│  ← CLICK!
└──────────────────┘

RESULT: ✓ Question appears in list below!
```

## 💡 Pro Tips

### Tip 1: Copy-Paste Answers
```
If answer is long, copy from source:
Question: The architect is ________
Answer: (copy from document) → Paste into Answer field
```

### Tip 2: Question Number Pattern
```
Listening Part 1: 1-10
Listening Part 2: 11-20
Listening Part 3: 21-30
Listening Part 4: 31-40

Use sequential numbering!
```

### Tip 3: Fill Multiple Questions Fast
```
No.  Question              Answer
[1]  The city is ____      Tokyo       → Click ➕
[2]  Population is ____    14 million  → Click ➕
[3]  Main industry is ____ Tourism     → Click ➕
```

### Tip 4: Delete & Redo
If you make a mistake:
- Click × next to wrong question
- Fill form again
- Click ➕ Add
