# 🚀 Quick Test Builder - INTEGRATED INTO ADD EXAM PAGE

## What Changed?

**The easy question builder is NOW BUILT INTO the `/admin/add-exam` page!** You no longer need to go to a separate page. Questions are added right where they belong.

---

## How to Use

### Step 1: Go to Add Exam Page
- Login as admin (admin@gmail.com / admin123)
- Click **"Add New Exam"** on Admin Dashboard
- Fill in Title, Description, Test Mode
- Click **"Next: Listening"** or **"Next: Reading"**

### Step 2: Create Question Groups
1. Each section/passage shows a button: **"+ Add New Group (e.g., MCQ, Matching)"**
2. Click it to add a new question group
3. A form appears with:
   - **Question Type Selector** - Choose from 5 types (Gap Filling, MCQ, T/F/NG, Matching, Diagram Labelling)
   - **Group Header** - e.g., "Questions 1-5"
   - **Type Description** - Shows what this question type does

### Step 3: ADD QUESTIONS EASILY - NEW! 🎉

**Now you see a QUICK QUESTION BUILDER at the top:**

```
┌─────────────────────────────────────────────┐
│ 📝 Easy Question Builder                    │
│                                             │
│ No.: [_____] Question Text: [___________]   │
│              Answer: [____] ➕ Add          │
└─────────────────────────────────────────────┘
```

Simply fill in:
1. **No.** - Question number (e.g., 1, 2, 3)
2. **Question Text** - The question or sentence (e.g., "The building was built in ____")
3. **Answer** - The correct answer (e.g., "1995")
4. Click **➕ Add** button

That's it! The question appears in the list below immediately.

### Step 4: Review & Remove Questions

Below the builder, you see all added questions:

```
1. The building was built in ____ [completion]
2. What is the main idea? [completion]
...
```

Each question has an **×** button on the right if you want to delete it.

### Step 5: Add More Questions for Different Types

Want MCQ instead? Follow the same steps:
1. Create a new group (click "Add New Group")
2. Select **"Multiple Choice (Single)"** as the type
3. The builder shows number + question + answer fields
4. Add questions the same way!

## Tips for Success

1. **Number Questions Sequentially** - Use 1-10 for first group, 11-20 for second, etc.
2. **Be Specific with Answers** - For completion: exact word (e.g., "vital", "2023")
3. **Use T/F/NG for Reading** - Great for comprehension questions
4. **Gap Filling for Listening** - Let students hear and type the answer
5. **MCQ for Both** - Very common in IELTS exams

## Troubleshooting

**❌ "Please fill in question text and answer"**
- Make sure both Question Text and Answer fields have content

**❌ Questions not appearing**
- Scroll down to see the questions list
- Make sure you clicked the ➕ Add button

**❌ Need to change a question type**
- Delete the group and create a new one with the correct type
- Or edit individual questions in the full editor below
